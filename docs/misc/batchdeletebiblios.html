NAME

batchdeletebiblios.pl

SYNOPSIS

./batchdeletebiblio.pl file1 [file2 ... filen]

This script batch deletes biblios which contain a biblionumber present in file passed in parameter. If one biblio has items, it is not deleted.

OPTIONS

-h|--help

prints this help message
