NAME

Koha::REST::Plugin::Objects

API

Helper methods

objects.find

    my $patrons_rs = Koha::Patrons->new;
    my $patrons = $c->objects->find( $patrons_rs, $id );

Performs a database search using given Koha::Objects object and the $id.

Returns undef if no object is found Returns the API representation of the requested object. It passes through any embeds if specified.

objects.search

    my $patrons_rs = Koha::Patrons->new;
    my $patrons = $c->objects->search( $patrons_rs );

Performs a database search using given Koha::Objects object and query parameters.

Returns an arrayref of the hashrefs representing the resulting objects for API rendering.

Warning: this helper adds pagination headers to the calling controller, and thus shouldn't be called twice in it.
