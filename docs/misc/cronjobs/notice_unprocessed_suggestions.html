NAME

notice_unprocessed_suggestions.pl - Generate notification for unprocessed suggestions.

The budget owner will be notified.

The letter template 'TO_PROCESS' will be used.

SYNOPSIS

notice_unprocessed_suggestions.pl [-h|--help] [-v|--verbose] [-c|--confirm] [--days=NUMBER_OF_DAYS]

OPTIONS

-h|--help

Print a brief help message

-c|--confirm

This flag must be provided in order for the script to actually generate notices. If it is not supplied, the script will only report on the patron it would have noticed.

--days

This parameter is mandatory. It must contain an integer representing the number of days elapsed since the last modification of suggestions to process.

-v|--verbose

Verbose mode.
