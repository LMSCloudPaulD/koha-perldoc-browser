NAME

addorder.pl

DESCRIPTION

this script allows to add an order. It is called by :

neworderempty.pl to add an order from an existing record or from nothing.
newordersuggestion.pl to add an order from an existing suggestion.

CGI PARAMETERS

All of the cgi parameters below are related to the new order.

ordernumber the number of this new order.
basketno the number of this new basket
booksellerid the bookseller the librarian has to pay.
existing
title the title of the record ordered.
author the author of the record ordered.
copyrightdate the copyrightdate of the record ordered.
ISBN the ISBN of the record ordered.
format
quantity the quantity to order.
listprice the price of this order.
uncertainprice uncertain price, can't close basket until prices of all orders are known.
branch the branch where this order will be received.
series
notes Notes on this basket.
budget_id budget_id used to pay this order.
sort1 & sort2
rrp
ecost
GST
budget
cost
sub
invoice the number of the invoice for this order.
publishercode
suggestionid if it is an order from an existing suggestion : the id of this suggestion.
donation
