NAME

C4::Heading::UNIMARC

SYNOPSIS

use C4::Heading::UNIMARC;

DESCRIPTION

This is an internal helper class used by C4::Heading to parse headings data from UNIMARC records. Object of this type do not carry data, instead, they only dispatch functions.
